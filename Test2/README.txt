# Bens Tattoos - Minimalistic Tattoo Website

## Overview
This is a minimalistic website for a tattoo artist, showcasing recent works, the artist's background, and providing contact details.

## Files
1. **index.html** - Contains the homepage with sections: Home, About, Gallery, and Contact.
2. **index.css** - Custom styles, animations, and Tailwind CSS utilities.
3. **script.js** - Placeholder for JavaScript if needed.

## How to Run
1. Clone the repository.
2. Open the `index.html` file in your browser.

## Technologies Used
- Tailwind CSS
- Vanilla HTML and CSS
